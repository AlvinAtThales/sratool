SRA Tool - a Security Risk Assessment Tool -

A Security Risk Assessment is a structured, repeatable method for evaluating the potential security risk of a product/service and assigning appropriate security activities.

It is recommended to perform this risk assessment with a Security expert.

Based on the risk determined by the SRA, a Set of activities will be suggested to help with the mitigation of the risk evaluated



The actors/stake holders

Process Owner Activity
Step 1 Project Owner​ Register the Project .
Step 2 Project Managers, Architects, Developers, Security Experts (Depending on the size of the project one person may have mutiple responsabilities) Complete and save the Security Risk Assessment for the registered project.
Step 3 Security engineer / Security Expert Assign activities based on recommended project security risk.
Step 4 Project Managers, Architects, Developers, Security Experts Complete the assigned security activities.
